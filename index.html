Proyectos

Características clave

Calculalo es una herramienta simple para resolver reglas de tres. Permite introducir A, B y C, calcula el resultado con dos decimales, muestra la explicación paso a paso y guarda un historial local en el navegador.

Filosfía del proyecto

El proyecto consiste en crear una página web que solucione un problema muy habitual en las páginas web dedicadas a la enseñanza. La mayoría de webs que nos encontramos hoy en día para resolver ecuaciones matemáticas se limitan a mostrar el resultado y con suerte te enseñan el paso a paso. La filosofía de este proyecto cumple una función importante para la formación de los jóvenes y ataca directamente a un problema actual que afecta a millones de estudiantes, la falta de motivación a la hora de aprender.
